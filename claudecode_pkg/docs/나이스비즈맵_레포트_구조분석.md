# 나이스비즈맵 레포트 구조 분석

**분석 대상:** 나이스비즈맵 '장량동·커피전문점' 레포트 (26장, 2026.07 캡처)
**목적:** 경쟁 서비스가 무엇을 어떤 형식으로 보여주는지 파악 → 명당 창업 리포트가 최소한 이보다 잘 나오게 하기 위한 기준선

---

## 1. 레포트의 큰 틀

상단에 **탭 7개**로 섹션을 나눈다. 각 섹션은 "이 지표를 왜 봐야 하는지" 한 줄 설명 박스로 시작하고, **데이터(차트+표) → 해석 카드(파란 박스로 '이런 전략이 중요해요')** 순서로 끝난다. 즉 모든 섹션이 `안내문 → 시각화 → 전략 코멘트` 3단 구조로 통일돼 있다.

**맨 첫 화면 = 요약 대시보드.** 수익구조(상위/하위 20% 매출), 핵심 고객층(주요 고객·고객 변화), 매출 집중 요일(최고/최저)을 카드로 먼저 보여주고 상세로 들어간다.

## 2. 섹션별 — 무엇을, 어떤 형식으로

### 1. 상권 개요 및 입지 특성
- **상권 유형** (주거형/상업형 등): 도넛 차트 + 구성비(주거지역 78% 등)
- **주변 인프라**: 관공서·교육기관·금융기관·버스정류장·지하철 **개수**를 아이콘 리스트로
- **전략 코멘트**: "주거형 상권 전략 — 고정 고객 유지, 저녁·주말 집중"

### 2. 인기 업종과 관심 업종 현황
- **매출 증가율 TOP 5**: 순위 + 가로 막대바(+621.5% 등)
- **점포 수 증가율 TOP 5**: 순위 + 가로 막대바
- **선택 업종 성장 단계**: '뜨는 업종'과 선택 업종의 단계(회복세 등)를 카드로

### 3. 업종 현황 (선택 업종의 상권 내 위치)
- **업종 비중**: 선택 업종 점포 수 / 전체 점포 수, 지역 vs **전국 평균** 비교 막대 + 표 (전국 평균 대비 +0.2%p)
- **점포수 변화**: 최근 6개월 월별 세로 막대 + 표 (전월/3개월 전 대비 증감)
- **해석 카드**: "점포 밀집도 = 경쟁 강도. 5~9%는 전략 여지, 10%↑는 재검토"

### 4. 업종 매출 현황
- **총 시장규모**: 6개월 월별 세로 막대 + 증감률
- **예상 매출**: 점포 평균 / 중위 50% / 상위 20% / 하위 20% 매출을 **4분면 카드**로 → 이어서 6개월 추이 꺾은선(상위·평균·하위 3선) + 표
- **이용 건수 / 결제 단가**: 6개월 막대(건수) + 꺾은선(단가) 복합 차트 + 표
- **해석 카드**: "고객도 늘고 1인당 지출도 늘었다 → 프리미엄 전략"

### 5. 방문 결제 고객 분석
- **성별·연령별**: 20~60대 좌우 분기 막대(남/여) + 표. **이번달/지난달/작년 대비 탭**으로 3시점 비교
- **요일별 매출 집중도**: 월~일 세로 막대 + 비중 표
- **시간대별 매출 집중도**: 06~24시 7구간 꺾은선(건수 비중·금액 비중 2선) + 표
- **해석 카드**: "매출 집중 시간대 재고·인력 배치, 고객층 바뀌면 메뉴·운영도"

### 6. 소비 및 메뉴 분석
- **가장 많이 팔리는 메뉴 TOP 3**: 메뉴명 + 평균단가·최저가·최고가·매출 비중
- **전월대비 인기 급상승 메뉴 TOP 3**: 메뉴명 + 단가 3종 + 판매 증가율(+63%p 등)
- **해석 카드**: "스테디셀러 vs 신메뉴 함께 봐야"

### 7. 비용과 수익 통계
- **영업비용과 영업이익**: 지역 평균 영업비용·영업이익 + **경상북도(광역) 대비** %
- **비용 구조 분해**: 식재료비·인건비·임차료·세금·대표자 인건비 등 항목별 금액과 비중, 지역 vs 광역 2열 비교 표 + 누적 바
- **해석 카드**: "수익성은 매출이 아니라 영업이익. 서울 대비 수익성 낮음 → 비용·메뉴 점검"

## 3. 형식·톤의 특징 (배울 점)

- **모든 수치에 '증감'을 붙인다.** 절대값 옆에 항상 전월·3개월·작년 대비 화살표(↑↓)와 %. "지금 좋다/나쁘다"가 아니라 "흐름"을 보여줌.
- **비교 기준선이 항상 있다.** 전국 평균, 경상북도(광역), 지난달, 작년. 단독 숫자를 거의 안 쓴다.
- **차트 아래 반드시 표.** 시각화로 인상을 주고 표로 근거를 준다.
- **섹션마다 해석 카드(파란 박스).** 데이터를 "그래서 뭘 하라는 건지"로 번역. 사장이 데이터 못 읽어도 행동 지침이 남음.
- **6개월 시계열**을 반복적으로 씀 (점포수·시장규모·매출·건수 모두).
- 순위형은 메달 아이콘 + 막대바로 직관화.

## 4. ★ 명당 리포트가 이보다 잘 나오려면

나이스비즈맵은 **강력하지만 결정적 약점이 있다: 전부 '과거·현재' 데이터이고, "이 자리에 무엇이 들어와야 하나"에 답하지 않는다.** 우리 리포트는 그들의 형식 완성도(비교 기준선·증감·해석 카드·시계열)를 최소한 따라가되, 그들이 못 하는 것을 얹어야 한다.

### 반드시 따라가야 할 기준선 (안 하면 초라해 보임)
- 모든 수치에 비교 기준(전국/경북 평균, 전월 대비) + 증감 화살표
- 차트 + 표 병행, 섹션마다 해석(전략) 코멘트
- 6개월 시계열로 흐름 제시
- 요약 대시보드를 맨 앞에

### 우리만 넣을 수 있는 차별 항목 (나이스비즈맵에 없는 것)
1. **미충족 수요 = 주민 선결제 투표.** 그들의 "과거 매출"이 아니라 "이 자리를 원하는 주민 N명, 1순위 업종". 이건 어떤 상권분석 서비스에도 없는 우리 고유 데이터.
2. **개업일 대기 고객 수.** "당신을 기다리는 132명" — 첫 손님 명단. 미래 지표.
3. **공실 단위 리포트.** 그들은 '동+업종' 단위, 우리는 '특정 빈 점포' 단위로 "이 자리에 이 업종" 추천 + 근거.
4. **AI 배치 근거.** 왜 이 업종이 이 공실에 배정됐는지 적합도 점수 분해(투표·면적·경쟁).
5. **경북 전역 비교.** 나이스비즈맵도 광역(경상북도) 비교를 하므로, 우리도 양덕동 vs 경북 비교를 넣으면 대등.

### 기획서 리포트 5섹션과의 접목
우리 리포트 5섹션(①추천 업종·근거 ②예상 초기비용 ③필요 인허가 ④주변 경쟁 현황 ⑤선결제 대기 고객)에서, ③④는 나이스비즈맵식 상권 데이터(경쟁점·비용 구조)로 채우고, ①⑤는 우리 고유 데이터(투표·배치·대기 고객)로 채운다. **즉 "그들의 상권 분석 + 우리의 수요 검증"을 한 장에 합치는 것이 명당 리포트의 목표 형태.** 상권 데이터는 소상공인 API(업종 구성·경쟁·매출)로, 미래 수요는 투표 데이터로 공급된다.

### 주의
나이스비즈맵의 매출·영업이익·결제단가 같은 정밀 수치는 카드 실거래 기반이라 우리가 동일 정밀도로 내긴 어렵다. **정밀도로 경쟁하지 말고, 그들이 아예 없는 축(미래 수요·공실 단위·대기 고객)으로 차별화**하는 것이 현실적이고 심사에서도 유리하다. 상권 통계 부분은 소상공인 API가 주는 범위에서 "출처 있는 값"으로 정직하게 채우고, 없는 값은 만들지 않는다.
